# Xアルゴリズム攻略：インプレッション最大化のメカニズム

## 1. 概要
X（旧Twitter）のアルゴリズムは、ユーザーがポジティブな反応（エンゲージメント）をする可能性が高い投稿を「おすすめ（For You）」に表示する仕組みです。
2023年に公開されたコードと、最新のGrokベースのAIモデル（x-algorithm）の情報を統合すると、インプレッション（Impu）を獲得するための**明確な勝ち筋**が見えてきます。

## 2. 最重要項目：エンゲージメントの重み付け
アルゴリズムの内部では、ユーザーの各行動に対して「スコア」が付けられており、この合計点が高いほど多くの人に拡散されます。特に2023年の「Heavy Ranker」の設定値は、現在のAIが何を重要視しているかを知るための最も具体的な指標です。

### 🚀 圧倒的に重要なアクション
| アクション | 重み（スコア） | 解説 |
| :--- | :--- | :--- |
| **リプライへの返信（会話成立）** | **75.0** | **最強のシグナル**です。投稿についたリプライに対して投稿者（あなた）が返信し、会話が成立すること。単なる「いいね」の150倍の価値があります。 |
| **リプライ** | **13.5** | リプライがつくだけでも非常に高評価です。「いいね」の27倍の価値があります。 |
| **プロフィールクリック＆いいね** | **12.0** | プロフィールを見に行きたくなるような投稿は高評価です。 |
| **リンククリック（2分以上滞在）** | **10.0** | 外部サイトへの誘導でも、ユーザーが戻ってこずにしっかり読んでいれば評価されます（逆に即離脱はマイナス評価の可能性大）。 |

## 3. インプレッションを爆発させるための戦略

### ① 会話をデザインする（Conversation is King）
アルゴリズムは**「会話」**を極端に優遇します。
*   **戦略**: 投稿文は「言い切り」で終わらせず、議論の余地やツッコミどころ、質問を残すことでリプライを誘発しましょう。
*   **運用**: ついたリプライには必ず返信してください。これだけでスコアが**75倍**跳ね上がります。フォロワーとのラリーが続くほど、投稿は強力に拡散されます。

### ② "SimClusters"を意識してニッチを攻める
Xは全ユーザーを「コミュニティ（SimClusters）」に分類しています。
*   **仕組み**: あなたのフォロワーが「何に興味がある集団か」によって、あなたの属性（Known For）が決まります。
*   **戦略**: 初期は発信ジャンルを1つに絞ってください（例：筋トレ、AI、節約など）。
    *   ジャンルを広げすぎると、どのクラスタにも属さないと判断され、おすすめに乗りづらくなります。
    *   「〇〇と言えばこの人」という状態を作ると、そのコミュニティ内で優先的に表示されます。

### ③ 滞在時間（Dwell Time）を稼ぐ
最新のAI（Grokベース）は、クリックなどの明確なアクションだけでなく、「ユーザーがその投稿でスクロールを止めたか？」「どれくらい長く見ていたか？」を重視します。
*   **戦略**:
    *   長文ポスト（テキスト量で滞在時間を伸ばす）。
    *   動画・画像・図解（視覚的に足を止めさせる）。
    *   ツリー形式（スレッド）で続きを読ませる。

### ④ "Tweepcred"（アカウントの信頼性）を高める
GoogleのPageRankのように、ユーザー自身の「信頼性スコア」が存在します。
*   **ポイント**: 「信頼性の高いユーザー（すでに影響力のあるユーザー）」からフォローされたり、リプライをもらったりすると、あなたのスコアも上がります。
*   **注意**: 質の低いフォロワー（Botやスパム）を大量に買っても逆効果です。

### ⑤ X Premium（旧Twitter Blue）への加入
コード上でも、認証済みユーザー（Blue/Premium）の投稿はブースト（拡散優遇）される傾向が確認されています。
*   **効果**: 「おすすめ」タブへの露出が増えるため、本気で運用するならほぼ必須です。

## 4. 最新アルゴリズム（AI/Grok時代）の変化
現在は、上記のような手動設定のルール（ルールベース）から、AIが全自動で学習するモデル（ニューラルネットワーク）へ移行しつつあります。
*   **「ハック」は通用しなくなる**: 「ハッシュタグを2個つければ伸びる」といった表面的なテクニックは意味を失いつつあります。
*   **本質への回帰**: AIは「ユーザーが本当に楽しんでいるか？」を学習し続けます。結局のところ、**「見ていて面白い」「役に立つ」「つい何か言いたくなる」**コンテンツを作ることが、最強のアルゴリズム対策となります。
